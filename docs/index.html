Voice Commands

Narration Control
  • skip_previous "Repeat" — Restart from beginning
  • fast_rewind "Back" — Rewind one chunk
  • fast_rewind "Back 3" — Rewind 3 chunks (1-10)
  • pause "Pause" — Pause narration
  • play_arrow "Play" "Resume" — Resume narration
  • fast_forward "Skip" — Skip one chunk ahead
  • fast_forward "Skip 3" — Skip 3 chunks (1-10)
  • skip_next "Stop" "Skip All" "End" — Skip to end
  • mic "Mute" "Unmute" — Toggle microphone
Save & Restore
  • save "Quick Save" — Save to default slot
  • restore "Quick Load" — Restore from default
  • save "Save [name]" — Save with custom name
  • restore "Restore [name]" — Load named save
  • delete "Delete [name]" — Delete named save
Common Game Commands
  • explore "North" "South" "East" "West" — Directions (or N/S/E/W)
  • explore "Northeast" "Northwest" "SE" "SW" — Diagonals
  • stairs "Up" "Down" "In" "Out" — Vertical/entry
  • visibility "Look" "Examine [thing]" — Observe
  • inventory "Inventory" (or "I") — Check items
  • touch_app "Take [thing]" "Drop [thing]" — Manipulate items
  • undo "Undo" "Wait" "Again" — Meta commands
  • text_fields "Print [text]" — Send literal text (bypass parsing)
App Commands
  • info "Status" — Read status bar aloud
  • help "App Help" — Show all app commands
  • lightbulb "Get Hint" — ChatGPT assistance
  • build "Repair" — Fix broken game state
  • exit_to_app "Quit" — Auto-save and exit
